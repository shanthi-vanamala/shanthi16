7.Disadvantages of python?

Ans. 1.slower Execution Speed:interpreted language,making it slower than compiled language like c++ or java.

2.memory consumption:pythons dynamic typing and garbage collection can lead to higher memory usage.

3.Global Interpreter Lock:The GIL restricts the execution of multiple threads at the same time.
